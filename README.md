# satellite-orbit-calculator
Расчёт положения спутника на круговой орбите, преобразование координат и 3D-визуализация.

## Назначение
Этот проект демонстрирует базовые принципы орбитальной механики и системной инженерии:
- Расчёт положения спутника в заданный момент времени.
- Преобразование из орбитальной плоскости в глобальную экваториальную систему координат (повороты на наклонение и долготу узла).
- 3D-визуализация траектории спутника и его положения.
- Вычисление подспутниковой точки с учётом вращения Земли.
  
## Теория

### Параметры орбиты
- Большая полуось: `a = R_earth + h`
- Период: `T = 2π √(a³ / μ)`
- Среднее движение: `n = 2π / T`

### Положение в плоскости орбиты (круговая)
- Истинная аномалия: `ν = ν₀ + n·t`
- Координаты: `x = a·cos(ν)`, `y = a·sin(ν)`, `z = 0`

### Преобразование в глобальную систему
Последовательные повороты:
1. Вокруг оси Z на угол аргумента перигея `ω` (для круговой орбиты `ω = 0`).
2. Вокруг оси X на угол наклонения `i`.
3. Вокруг оси Z на угол долготы восходящего узла `Ω`.

### Подспутниковая точка
- Широта: `φ = arcsin(Z / r)`
- Долгота (с учётом вращения Земли): `λ = atan2(Y, X) - ω_earth·t`
